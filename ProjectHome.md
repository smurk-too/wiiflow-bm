a wiiflow mod by 91wii.com forum id book . This mod can replace System Menu of Wii (work together with autoboot feature by priiloader).
```
==== Feature list ====
Exit game return to wiiflow do not need priiloader (new)
Launcher for wiiware and system channels (press [Home] then [+] to enter 
channel list)
Launcher for Priiloader (0.3c above)
Launcher for system menu (need priiloader 0.3c above when set autoboot to 
wiiflow)
Launcher for HBC
Shutdown Wii
Restart Wii
Halt Wii
Using USB Keyboard, NGC Pad, Classic Pad control Wiiflow
Turn off WPad 2,3,4
Fat16 Support

Note: You need unlock your wiiflow to using launch system 
menu/priiloader/HBC
edit channels.ini to hide some channels or wiiwares on lock status.


　　Press [Home] in Wiiflow R42 BMr3, instead exit wiiflow immediate, it 
will display the following menu
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
	Up: Parent unlock.
	Down: Shutdown Wii.
	Left: Pause Wii.
	Right: Restart Wii.

	+: launch Channels (wiiware/vc/etc).

	A: continue original home key action.
	
	-: launch System Menu.
	1: launch HBC.
	2: launch Priiloader r69+
	
	Home or B: Back.
```
